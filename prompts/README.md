# prompts/

Cross-cutting, **platform-wide** prompt templates and prompt-engineering assets
shared by more than one layer.

Layer-specific prompts live with their layer (e.g.
`requirement_intelligence/prompts/`). Promote a prompt here only when it is
reused across layers.

Conventions:
- One prompt per file, versioned as text/markdown for reviewable diffs.
- Document placeholders at the bottom of each template.
